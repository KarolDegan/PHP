## 🎯 Visão Geral

Este repositório documenta meu aprendizado em PHP com foco no desenvolvimento backend, aplicando conceitos fundamentais e intermediários da linguagem.

O conteúdo inclui estudos sobre orientação a objetos, acesso a banco de dados, consumo de APIs, web scraping e boas práticas de arquitetura de software.

### 🧾 Gerenciador de Alunos (PDO + SQLite)

Sistema CRUD para gerenciamento de alunos e turmas com persistência em banco de dados.

* Cadastro de alunos
* Listagem e remoção

* Persistência com SQLite
